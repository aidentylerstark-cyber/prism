# Prism — Branding Assets

Source art and export scripts for Prism's visual identity. See
[../docs/tpv/branding.md](../docs/tpv/branding.md) for where each asset maps into the viewer
tree and the M1 rebrand checklist.

## Layout (to be populated in M1)
```
branding/
  icon/        source app icon → export to indra/newview/res/*.ico, macOS *.icns, Linux PNGs
  splash/      login splash / loading art
  wordmark/    Prism wordmark + monochrome mark (docs, site, About floater)
  export.md    notes on regenerating platform assets from source art
```

## Rules
- No Second Life Eye-in-Hand logo as Prism's mark (TPV §5); it appears only where the real SL
  login splash requires it.
- Channel string: `Prism Release` (see branding.md).
